# Implementation
## 1. Starting
t-code: `SMARTFORMS`
This t-code is used to create and manage Smart Forms, including Smart Styles and text modules.
- **Form:** Builds the form (structure, data layout, etc). **Prefix: ZSF.**
- **Style:** Defines text formatting styles (paragraph and character formatting) which can be applied across forms. **Prefix: ZSS.** [For Smart Styles, specific T-Code SMARTSTYLES can be used as well]
- **Text Module:** Creating reusable standard text blocks. **Prefix: ZTM.**
