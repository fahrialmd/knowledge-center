# Purpose
Smartstyles is used to create paragraph and character format.
- **Character formats** apply **style attributes** such as bold, italics, underline, font type, size, color, and effects like superscript or subscript to specific characters or words within a paragraph. It is used to style individual characters or sections of text. Character formats can also be used to define bar codes.
- **Paragraph formats** define the **overall layout** of an entire block of paragraph, contains the attributes of a character format with additional layout properties like indents, alignment, line spacing, spacing before and after, tabs, numbering, outlining, and page-break behavior.
In other words, paragraph formats define the whole text section, while character formats define specific text section.
